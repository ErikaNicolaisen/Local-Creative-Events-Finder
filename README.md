# StageUp Copenhagen

A website created with the intention of gathering Copenhagen-based volunteer 
opportunities within sound, lighting, photography, and stage management in one place. 
The goal is to make it easier for people interested in these fields to find relevant 
experience, build their CVs, and expand their professional network within the live 
music and events industry.

## How to access the website
Click the link below to open the website:
