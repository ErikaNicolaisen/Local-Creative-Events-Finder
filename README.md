# StageUp Copenhagen

A website created with the intention of gathering Copenhagen-based volunteer 
opportunities within sound, lighting, photography, and stage management in one place. 
The goal is to make it easier for people interested in these fields to find relevant 
experience, build their CVs, and expand their professional network within the live 
music and events industry.

## Prerequisites

Before running the project, make sure you have **Node.js** installed on your computer.

## Getting Started

### 1. Clone the repository
```
git clone https://github.com/ErikaNicolaisen/Local-Creative-Events-Finder.git
```

### 2. Navigate to the project folder
```
cd Local-Creative-Events-Finder/events-finder
```

### 3. Install dependencies
```
npm install
```

### 4. Start the server
```
node server.js
```

### 5. Open the website
Open your browser and go to:
```
http://localhost:3000
```

The website should now be running locally on your computer.
